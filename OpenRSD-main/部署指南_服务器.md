## 环境要求

| 项目 | 要求 |
|------|------|
| Python | 3.8 ~ 3.10 |
| CUDA | ≥ 10.2 (推荐 11.8 / 12.1) |
| PyTorch | ≥ 1.12 (推荐 2.1+) |
| GPU 显存 | ≥ 8GB (推荐 24GB) |
| 系统 | Linux (推荐) / Windows |

## 快速安装

### Linux
```bash
cd OpenRSD-main
bash install.sh
```

### Windows
```cmd
cd OpenRSD-main
install.bat
```

### 手动安装（如果脚本失败）
```bash
# 1. 创建环境
conda create -n openrsd python=3.10 -y
conda activate openrsd

# 2. PyTorch
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121

# 3. mmcv (从仓库自带zip)
unzip mmcv-main.zip -d /tmp/mmcv
cd /tmp/mmcv/mmcv-main
pip install -e . --no-build-isolation
cd -

# 4. 安装 OpenRSD
pip install -v -e .
```

## 训练自己的数据

### 数据格式（DOTA格式）
```
data/your_dataset/
├── images/
│   └── img1.png
└── annfiles/
    └── img1.txt   # x1 y1 x2 y2 x3 y3 x4 y4 category difficult
```

### 完整流程
```bash
# Step 1: 图像切片（大图→1024×1024 patches）
python tools/data/your_dataset/split/img_split.py --base_json config.json

# Step 2: 提取 DINOv2 视觉特征
python M_Tools/Data2_DIOR_R/Step4_Extract_DINOv2_Embeds_8_3.py

# Step 3: 生成 support 数据（文本+视觉embeddings）
python M_Tools/Data2_DIOR_R/Step5_3_Prepare_Visual_Text_DINOv2_support.py

# Step 4: 训练
python tools/train_rotate.py ${CONFIG} --gpu-id 0
```

## 三类检测头的使用场景

| 模式 | 配置文件 | 用途 |
|------|----------|------|
| **对齐头** | val_using_aux=False | 实时推理，支持大词汇表 |
| **融合头** | val_using_aux=True | 高精度场景 |
| **HBB头** | HBB 后缀配置 | 水平框检测 |
