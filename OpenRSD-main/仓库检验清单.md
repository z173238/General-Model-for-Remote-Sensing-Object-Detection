## 已知问题

1. **environment.yml 是 Linux + CUDA 10.2**，Windows/高版本CUDA需手动适配
2. **mmcv 需要特殊安装**，仓库提供了 mmcv-main.zip
3. **SimpleRun 硬编码了路径**：`proj_root = '/workspace/OpenRSD'`，需改为实际路径
4. **百度网盘依赖**：所有权重和数据集都在百度网盘，GitHub 只有代码
5. **预训练需 SkyCLIP**：`M_Tools/Data2_DIOR_R/Step5_3_*` 脚本需要 SkyCLIP ViT-L/14 权重

## 已验证的命令入口

```bash
# 训练
python tools/train_rotate.py ${CONFIG}
python tools/train.py ${CONFIG}          # 水平框

# 测试
python tools/test_rotate.py ${CONFIG} ${CKPT}
python tools/test.py ${CONFIG} ${CKPT}   # 水平框

# 批量训练
cd EXP_CONFIG
python multi_train_any_gpu.py -k A08 -c 10 -d 0,1,2,3

# 批量评估
cd M_Tools/Eval_Tools
python eval_diff_epochs.py -d 0 -e 24

# 快速推理
python SimpleRun/step1_inference.py

# 权重格式转换
python SimpleRun/step2_trans_to_weights_only.py
```

## 建议

1. 先下载**轻量版百度网盘包**（仅CKPT+Embeddings），用 SimpleRun 跑通推理验证环境
2. 确认环境 OK 后再下载完整数据集，考虑训练/微调
3. SimpleRun/step1_inference.py 中把 `proj_root` 和 `gpu` 改成服务器的实际值
